# Site Vision: David Fincher Social Network

**Project ID:** 2574078534653431404

## 1. Goal
A high-tension, cinematic social network that feels forensic, highly analytical, and stylishly grim.

## 2. Sitemap
- [ ] index.html (Main Social Feed)

## 3. Roadmap
- `index.html`: The main feed page featuring posts, a stark sidebar, and the grainy animation.
- `profile.html`: User profile page with forensic stats and grid-like history.
- `network.html`: A node-based visualization of user connections.

## 4. Creative Freedom
- Easter eggs related to classic Fincher films in the UI microcopy.
