An Analysis Dashboard with a Chat UI for interrogating the repository.

**DESIGN SYSTEM (REQUIRED):**
Use a David Fincher-inspired moody, cinematic aesthetic. The background should be Abyss Black (#090a0a) with surfaces in Sterile Charcoal (#131615). Use clinical, technical typography (Inter/Mono) with crisp, sharp edges (rounded-none or sm) and 1px borders (#1c2522). The primary accent color is a muted, sickly neon yellow/green (#d2d854) for sparse interactive highlights. Text should be clinical white (#e0e5e3) and washed mint (#8b9e97). Maintain a strict, grid-based, high-density layout. Include a CSS-based light film grain noise overlay throughout the site.

**Page Structure:**
1. A stark, minimal header with a text logo "THE NETWORK" in monospace, and an active repository status indicator (e.g. "facebook/react [INDEXED]").
2. A left sidebar acting as a 'Control Panel' with links (Feed, Analysis, Nodes) and a list of key detected architectures.
3. The main area: A highly technical Chat UI for "interrogating" the codebase. The chat history should look like forensic logs. The user messages align right, the AI responses align left inside sharp 1px bordered boxes.
4. The chat input bar at the bottom should be a harsh outlined input field with a glowing active state and a sharp "EXECUTE" button.
5. A right sidebar with 'Repository Analytics' showing data density, top contributors, or file types in a stark, clinical style.
